# Changelog

## v0.1.0-english.1

First public derivative release of the English-maintained version.

Included in this release:

- English-oriented derivative repository packaging
- public attribution and derivative notice
- Moonshot / Kimi provider endpoint update to `api.moonshot.ai`
- compatibility fix for Kimi K2.5 temperature handling
- removal of internal handoff and local work notes from the public repo

Known limitations:

- self-hosted setup is still required
- external service keys are required for non-local providers
- public release is source-first rather than packaged SaaS
